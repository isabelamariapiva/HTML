Aplicar o mesmo estilo para h1, h2 e h3
sem repetir o código?

Para aplicar o mesmo estilo a h1, h2, h3, use o seletor de grupo
em CSS, como h1, h2, h3 { /* estilos aqui */ }. Isso permite que
você defina um único conjunto de regras de estilo para todos os
três elementos de uma vez, como cor, tamanho da fonte e margem.

h1, h2, h3 {
color: #333;
font-family: Arial, sans-serif;
}
