A "Receita" do CSS

O que é um Seletor?

Seletores CSS são padrões que identificam e
selecionam elementos HTML para que eles
possam ser estilizados.

O que é uma Propriedade?

É uma característica de um elemento HTML, como
color para cor de texto, ou font-size para
tamanho da fonte (background, display, position).

O que é um Valor?

É a parte de uma declaração que define a
configuração ou aparência exata da
propriedade à qual está associado. Ele fornece
a instrução específica para a propriedade. Os
valores podem assumir muitas formas.
